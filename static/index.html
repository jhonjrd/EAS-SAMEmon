Configure Webhook in Home Assistant
×

Follow these steps to create a Home Assistant automation triggered by EAS-SAMEmon alerts:

  1. In Home Assistant, go to Settings → Automations & Scenes.
  2. Click Create Automation → Create new automation.
  3. Under Triggers, click Add Trigger and select Webhook.
  4. A Webhook ID is generated automatically — you can also type a custom one (e.g. eas-samemon-alert).
  5. Enable Only accessible from the local network if EAS-SAMEmon runs on the same LAN.
  6. The URL to paste in EAS-SAMEmon is:
    http://<HA-IP>:8123/api/webhook/<webhook-id>
  7. Add your desired actions (notification, alarm, script…), then Save. Paste the URL in EAS-SAMEmon, enable the toggle and save.

Sigue estos pasos para crear una automatización en Home Assistant disparada por alertas de EAS-SAMEmon:

  1. En Home Assistant, ve a Configuración → Automatizaciones y Escenas.
  2. Haz clic en Crear automatización → Nueva automatización.
  3. En Disparadores, haz clic en Agregar disparador y selecciona Webhook.
  4. Se genera un ID de Webhook automáticamente — también puedes escribir uno personalizado (ej. eas-samemon-alert).
  5. Activa Solo accesible desde la red local si EAS-SAMEmon corre en la misma red.
  6. La URL que debes pegar en EAS-SAMEmon es:
    http://<IP-HA>:8123/api/webhook/<webhook-id>
  7. Agrega las acciones deseadas (notificación, alarma, script…), luego Guarda. Pega la URL en EAS-SAMEmon, activa el interruptor y guarda.
